In this project you can find the scripts and minimal data for the manuscript: 'IL-22 induces Paneth cell metaplasia in the colonic epithelium of ulcerative colitis, promoting wound healing via REG3A'
from Tomohiro Muto1, Go Ito et al. 

RNAseq_Il22_REG3A.R refers to the script from bulk RNaseq experiment: RNA-seq analysis comparing Reg3afl/fl and Reg3aÎ”IEC colonic organoids treated with IL22.

Spatial_IL22_REG3A refers to the script from spatial RNaseq experiment: spatial RNAseq analysis with Visium HD for paired ileum and colon samples from to UC patients.
